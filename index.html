Overview

The Peak Climbing Club has conducted a survey to review the current joining procedure to see if there are any improvements that can be made to the process.

Methods

This report was generated using Quarto and the R statistical programming language and is hosted on GitHub Pages.

Did you decide to join the Peak Climbing Club?

Do you have any feedback about how the club could improve the joining process?

improve
I’m wary about pairing up and not finding the group.
Give prospective members access to Whats App
- Some way of knowing whether or not meets are actually happening, confirmation that someone else will be there.
- A single point of contact for new members, I was given 4 phone numbers I could use to organise going to the first meet, which seems redundant?
give prospective members access to the whatsapp groups. giving 4 random people’s numbers that they don’t respond to is suboptimal.
The club was easy to find at Horseshoe but I may have struggled if it was busier, maybe a small banner might be good?
I found the form very clear. I wasn’t 100% sure at what point I should or had to commit to joining.
The first two points can be improved. Please see my previous point
